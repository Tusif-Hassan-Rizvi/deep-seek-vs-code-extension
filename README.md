# Create VS Code Extension

```*Note: Before running this project, download and install Ollama on your system from here:```

```bash
https://ollama.com/download
```

### If you want to setup new project
Step 1

```bash
npm install -g yo generator-code
```

Step 2

```bash
 yo code
```

### Use this project

Step 3

```bash
npm install
```
